# Revolut Design Patterns

## Key Visual Elements

### Color Palette
- **Background:** Clean white (#FFFFFF) or dark mode (#0D0D0D / #1A1A1A)
- **Primary accent:** Blue (#0066FF) for CTAs and links
- **Text:** Black (#000000) for primary, gray (#666666) for secondary
- **Cards:** Subtle shadows, rounded corners (16-20px radius)

### Typography
- **Balance/Amount:** Very large, bold (40-48px), with smaller decimals
- **Currency label:** Small, muted text below balance
- **Section headers:** Small caps or uppercase, muted color
- **Transaction text:** Medium weight, clean sans-serif

### Layout Patterns
- **Top navigation:** Horizontal tabs (Home, Cards, Stocks, Crypto, Vaults)
- **Balance hero:** Large centered amount with currency flag
- **Quick actions:** Horizontal row of pill buttons (+ Add money, â†’ Send, ...)
- **Transaction list:** Icon + Name + Time on left, Amount on right
- **Cards:** Rounded corners, subtle shadows, colorful promo cards

### Spacing
- **Generous whitespace** between sections
- **Consistent padding:** 16-24px
- **Card padding:** 20-24px internal

### Interactive Elements
- **Buttons:** Pill-shaped with light background (blue tint)
- **Icons:** Simple line icons, consistent stroke weight
- **Bottom nav:** Icon + label, active state highlighted

### Dark Mode Variant
- Deep black background
- Gradient hero card (blue/purple)
- White text on dark
- Subtle glass effects

## Application to UltraWealth

1. **Account Overview:** Large balance hero, horizontal quick actions
2. **Goals Dashboard:** Cards with progress, clean list layout
3. **Goal Detail:** Large value display, clean performance metrics
4. **Trade Flow:** Simple forms, clear CTAs
